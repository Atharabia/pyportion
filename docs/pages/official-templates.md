# Official Templates

PyPortion maintains a set of official templates that you can download and use right away.

| Template | Description | URL |
|---|---|---|
| `cli-template` | A ready-to-use structure for building Python CLI applications | `https://github.com/Atharabia/cli-template` |

## Download an Official Template

<div class="termynal" data-termynal>
    <span data-ty>portion template download gh/Atharabia/cli-template</span>
</div>

For more details on managing templates, see the [Template Management](user-guide/3-templates-management.md) guide.
